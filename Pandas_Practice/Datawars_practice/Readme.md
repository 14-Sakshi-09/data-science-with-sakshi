# DataFrames Practice - Working with English Words

This project focuses on practicing Pandas DataFrame operations using a dataset of English words from DataWars.

## Topics Covered
- Reading CSV files using Pandas
- DataFrame indexing
- Filtering and selection
- Aggregation functions
- Creating new columns
- Data analysis using Pandas

## Technologies Used
- Python
- Pandas
- Jupyter Notebook
- Google Colab

## Tasks Performed
- Found total number of words in the dataset
- Calculated maximum word values
- Filtered words using conditions
- Created a new Ratio column
- Identified words with highest ratios
- Performed DataFrame exploration and analysis

## Files Included
- `Project.ipynb` → Main notebook
- `functions(df).txt` → Notes and Pandas functions used
- `README.md` → Project documentation

## Dataset
Dataset (`words.csv`) was provided inside the DataWars lab environment.
